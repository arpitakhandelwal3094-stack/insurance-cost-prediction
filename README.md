# Insurance Cost Prediction

## Problem Statement
Predict medical insurance charges based on customer data.

## Dataset
- Size: 1,338 records and 7 features

## Tools & Technologies
Python, Pandas, NumPy, Scikit-learn

## Model Used
Linear Regression

## Model Performance
- RMSE: 4295.22
- R² Score: 0.88

## Key Insights
- Smoking and age significantly increase insurance costs
- BMI also impacts insurance charges

## Conclusion
The model provides a reliable estimation of insurance costs and highlights key influencing factors.
